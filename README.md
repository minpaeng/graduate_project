# 필요한 기능
### 전제조건
* done 명령어를 입력하면 예상 약속 시간과 함께 시각화 화면을 볼 수 있는 URI(URI 파라미터로 그룹 ID, 날짜 리스트를 넘겨받아야 함)가 제공된다.
* 약속시간 예시) 2022년 6월 4일 0:0~24:0

 <br/>

### 가장 먼저 해야할 것
* 출력된 값이 1개 이상일 때: 출력된 약속시간 값(1~10개) 달력에 랜더링(순위를 함께 표시)
* 사용자가 1순위 일정에 대한 구글 캘린더에 예약을 원할 때: 이메일을 입력할 수 있는 텍스트박스를 제공하여, 이메일 입력 시 사용자 구글 캘린더에 일정 추가

<br/>

### 위 기능 완성 시 고려해볼만한 기능
* 모바일 화면 고려하여 html 수정
* 웹 URL을 제공하지 않고, 봇을 통해 달력 캡쳐 사진을 채팅방에 전달하고 \
  1순위 일정에 대해 구글 캘린더 예약을 원하는 사용자에게 이메일 주소를 받아 예약을 해줌
